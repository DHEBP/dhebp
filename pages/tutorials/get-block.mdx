## Get Block

Returns all the information of the requested block.

#### Available parameters

Certainly! Here's a Markdown table based on your specifications:

| Name   | Type    | Mandatory | Comment       |
|--------|---------|-----------|---------------|
| hash   | Hash    | false     | Block hash    |
| height | uint64  | false     | Block Height  |

#### Body

```json
{
    "jsonrpc": "2.0",
    "id": "1",
    "method": "DERO.GetBlock",
    "params": {
        "height": 420
    }
}
```

#### cURL Request

```bash
curl -X POST \
  http://127.0.0.1:40402/json_rpc \
  -H 'content-type: application/json' \
  -d '{
    "jsonrpc": "2.0",
    "id": "1",
    "method": "DERO.GetBlock",
    "params": {
        "height": 420
    }
}'
```
