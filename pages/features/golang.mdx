## Golang

![Golang](/assets/gologo.png)

DERO has been developed from scratch in Golang and is being actively developed by the original developers. It is not a fork of any other coin.

Golang, was chosen be the creators of Dero for a variety of factors:

1. **Simple to Learn:** Golang was designed to be straightforward. It's like using a set of clear, easy-to-understand instructions to build things. This simplicity makes it easier for new programmers to learn and start building cool stuff.

2. **Efficient and Fast:** When you're building something, speed matters, right? Golang is super fast! It helps programmers create software that runs quickly and smoothly. This speed makes applications built with Go feel snappy and responsive.

3. **Concurrent and Scalable:** Imagine juggling multiple tasks at once without dropping anything. Golang helps software handle lots of tasks simultaneously without getting confused or slowing down. This is called concurrency. It's like having many workers efficiently doing different jobs at the same time.

4. **Great for Web Services:** Ever used apps like YouTube, Gmail, or Uber? Golang is great for building similar kinds of applications, especially ones that need to handle lots of users at the same time. It's like having a super strong foundation to support thousands or even millions of people using an app at once.

5. **Built-in Safety and Reliability:** Golang helps programmers write more stable and reliable software. It's like having a safety net to catch mistakes before they cause big problems. This means fewer bugs and more trustworthy software.

6. **Open Source and Community Support:** Imagine having a community of friends helping you build something. Golang is open source, which means it's free to use and lots of smart people worldwide are continuously improving it. This helps developers by providing tools, support, and guidance.

Go is a powerful programming languate that makes it easier for programmers to build software quickly, reliably, and efficiently. Its simplicity, speed, ability to handle lots of tasks, and support for building web services make it a valuable and popular choice among developers.
