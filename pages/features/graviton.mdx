# Graviton
![Graviton](/assets/graviton.png)
Graviton provides a robust, feature-rich, and corruption-resistant key-value store database suitable for various applications and environments.

The Graviton approach ensures data integrity, supports snapshots for easy backups, and allows live updates without risking database corruption.

At the core of Dero's infrastructure lies Graviton, an innovative key-value store database crafted specifically to meet the intricate needs of DeroHE. Functioning as Dero's database, Graviton plays a pivotal role in securely storing all account balances in an encrypted state. It draws inspiration from the article's comparison of being "ZFS for key-value stores," an apt representation of its ability to provide robustness and efficiency akin to ZFS, a file system renowned for its integrity and data protection features.

Analogous to a bank, Graviton uniquely updates balances in an encrypted manner, relying entirely on cryptographic proofs to maintain accuracy, ensuring unparalleled privacy and security for account holders. However, what truly elevates Graviton as Dero's beating heart is its capability to store uncompiled code within its database structure, eliminating the necessity for a centralized code source.

This groundbreaking feature empowers the entire Dero network with resilience against censorship attempts, even in scenarios where an entity attempts to shut down the entire internet. The decentralized nature of Graviton enables the restoration of the entire Dero network from a single node copy, rendering it impervious to disruptions or attempts at censorship.

Graviton embodies more than a database; it represents a state-of-the-art, authenticated, and append-only key-value store. Its outstanding features, including version tracking, snapshots, cryptographic proofs, and efficient diffing mechanisms, address critical limitations observed in traditional databases.

Designed for reliability, security, scalability, and user-friendliness, Graviton emerges as an indispensable choice for applications requiring specialized functionalities within a key-value store. Its open-source nature, seamless integration into Golang applications, and unwavering commitment to quality code further solidify its significance in the database landscape. With an essence reminiscent of "ZFS for key-value stores," Graviton stands as a testament to innovation, fortifying Dero's core infrastructure and safeguarding its network with unparalleled resilience.

### Append-only data storage system
An append-only data storage system in blockchain ensures data immutability, security, and transparency. 

It enables tamper-proof records by allowing only the addition of new data without altering existing information.

### Graviton API
Graviton's minimalistic API, scalability for exabyte-sized data stores, decoupled storage layer, cryptographic proof generation, versatile storage options, and complete code coverage make it a reliable and adaptable database solution suitable for various applications.

Graviton's codebase is available on GitHub (https://github.com/deroproject/graviton)
